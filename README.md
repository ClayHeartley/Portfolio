# Clayborne Heartley Portfolio Website

A static tech/media portfolio built with HTML, CSS, and JavaScript.

## How to use
1. Open `index.html` in your browser.
2. Replace placeholder sections with additional screenshots, links, or a headshot.
3. Upload the folder to GitHub Pages, Netlify, or Vercel.

## Customize
- Edit text in `index.html`.
- Edit colors and layout in `styles.css`.
- Add new animation behavior in `script.js`.

## Suggested additions
- Add a professional headshot to `assets/` and place it in the hero section.
- Add more UAlbany Athletics, Trend, and Clays Kicks screenshots to the campaigns section.
- Convert the resume DOCX to PDF before publishing if you want easier browser viewing.
